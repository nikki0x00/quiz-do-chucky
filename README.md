Um quiz interativo em linguagem C sobre o universo do Chucky, com 3 níveis de dificuldade (fácil, médio e difícil), cada um contendo 5 perguntas. O jogador responde em minúsculas e o programa verifica as respostas automaticamente usando strcmp, exibindo a pontuação final de 0 a 5.

O código utiliza vetores de strings para armazenar perguntas e respostas, além de estruturas básicas da linguagem como switch, loops e funções.
